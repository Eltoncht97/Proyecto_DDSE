# Esquema de la presentación (.pptx)

Guía de diapositivas para la sustentación grupal. Exportar a **.pptx**.

1. **Portada** — Título del proyecto, curso, integrantes y coordinador.
2. **Problemática** — Gestión manual de pedidos/ventas en un restaurante.
3. **Objetivos (SMART)** — OBJ 1 y OBJ 2.
4. **Alcance** — Módulos incluidos y fuera de alcance.
5. **Arquitectura** — Diagrama multicapa (Presentación / Negocio / Datos / Entidades + SQL Server).
6. **Tecnologías** — C#, .NET Framework 4.5, Windows Forms, ADO.NET, SQL Server, LINQ, ReportViewer.
7. **Modelo de datos** — Diagrama de tablas y relaciones.
8. **Demostración** — Login → Mantenimientos → Registrar pedido → Facturar → Reporte de ventas.
9. **Temas del curso aplicados** — Mapeo Temas 1–6 (ver README, sección 6).
10. **Conclusiones y recomendaciones.**
11. **Cierre / Preguntas.**

> Sugerencia (Tema 6 / diseño): tipografía estándar (Segoe UI/Arial/Calibri), colores
> institucionales, jerarquía visual y consistencia entre diapositivas.
